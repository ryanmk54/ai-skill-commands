# Local Config

Copy this file to `LOCAL_CONFIG.md` and update the values for your machine.

## Board Root

`/home/your-user/path/to/personal-kanban`

## Status Folders

- `todo`
- `in-progress`
- `review`
- `done`
- `archive`

## Notes

- The board root should contain one folder per status.
- The folder name is the source of truth for task status.
- Keep `LOCAL_CONFIG.md` untracked if you plan to publish this repository.
